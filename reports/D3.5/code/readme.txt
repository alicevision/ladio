LADIO - Live Action Data Input / Output
EU H2020 Project No. 731970
Deliverable D3.5: Lens calibration from 3D Reconstruction in openMVG
Tomas Pajdla, pajdla@cvut.cz, 2017-04-10

Ladio-D35-Literature/		- Literature
Ladio-D35.maple			- Maple 2016 code for developing minimal solver for estimating absolute pose with unknown focal length and radial distortion
Maple/				- Maple utilities
Ladio-D35-Matlab/		- Matlab code of minimal solvers, their demonstration and testing

Ladio-D35-Matlab/data/		- data for testing minimal solvers
Ladio-D35-Matlab/bin/ladio	- Matlab 2016 code.

To run Matlab code:

Run Matlab
>>cd Ladio-D35-Matlab/bin/ladio
>>ladio
